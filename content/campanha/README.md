# Campanha – Anúncios Instagram

Pasta com textos em Markdown para a **campanha de anúncios no Instagram** — objetivo: **atrair novos clientes** que queiram solicitar página, site ou sistema sob medida.

Cada post funciona como **anúncio**: mostra um trabalho realizado e convida o público a **solicitar a página** (orçamento / projeto) para o negócio dele.

## Convenção dos arquivos

- Nome: `[projeto ou tema]-[descrição-curta].md`
- Cada `.md` contém: título, legenda (tom de anúncio), CTA para solicitar página/orçamento, sugestão de mídia e hashtags.
