# Prints para os posts – Fortão Prêmios

Coloque aqui os prints do site **fortopremios.com.br** para usar nos posts da campanha.

## O que capturar

### 1. Página principal (site público)
- **Arquivo sugerido:** `fortao-pagina-principal.png` (ou .jpg)
- **URL:** https://fortopremios.com.br
- **O que mostrar:** home com rifas em destaque, lista de rifas, botões de compra, layout responsivo.

### 2. Painel administrativo
- **Arquivos sugeridos:**
  - `fortao-painel-dashboard.png` — tela do dashboard com estatísticas
  - `fortao-painel-rifas.png` — gerenciamento de rifas
  - `fortao-painel-clientes.png` — lista de clientes (se aplicável)
- **URL:** https://fortopremios.com.br/admin (após login)
- **O que mostrar:** dashboard, gráficos, tabelas, menus do painel.

Use estes arquivos nas artes do carrossel/feed (capas e slides) conforme o post em `fortao-premios-trabalho-realizado.md`.
